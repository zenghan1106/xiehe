🚨 危重程度分级处理
崩溃气道 / 气管插管指征 → 球囊面罩通气

危（SpO₂ < 80%） → 储存氧面罩 / 高流量氧疗 / 无创通气

重（80% < SpO₂ < 88%） → 面罩给氧或鼻导管

非危重 → 直接进入ESCAPE评估
🎯 目标达成检测
SpO₂ 94%~98% 且 生命体征稳定

否 → 返回 球囊面罩通气（重新评估）
是 → 进入 ESCAPE & 评估
🔍 ESCAPE评估
ESCAPE高危因素

是 → 目标SpO₂ 88%~93%
（滴定氧浓度：无创/高流量装置或选择文丘里面罩）

否 → 维持 SpO₂ 94%~98%
📊 CO₂监测
监测CO₂

CO₂进行性增高：
是 → 无创通气 → 返回病情评估
否 → 病情评估 / 继续或终止氧疗
📝 病情危重程度判定标准
SpO₂ < 80% 为危； 80% < SpO₂ < 88% 为重
根据病情选择高浓度或低浓度氧疗工具
⏱️ 动态评估要求
氧疗开始后应每5~10分钟评估患者SpO₂变化情况
• 若SpO₂未能上升至目标范围，应积极寻找原因并行血气分析检查全面评估患者情况
• 若SpO₂上升至目标范围内，存在ESCAPE高危因素应在30~60分钟内复查血气了解血CO₂
